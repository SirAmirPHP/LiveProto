# рЅ┤рѕїрїЇрѕФрѕЮ рѕЎрІџрЅЃ ріарІЇрЅ│рѕе рѕўрѕерЅЦ рЅдрЅх

рІГрѕЁ рЅ┤рѕїрїЇрѕФрѕЮ рЅдрЅх ріе Instagram Reels рѕЎрІџрЅЃріЋ рѕЏрІѕрЅЁ рІГрЅйрѕІрѕЇрЇб рЅ░рїарЅЃрѕџрІјрЅй Instagram Reels ріарїѕріЊріърЅйріЋ рѕірѕІріЕ рІГрЅйрѕІрѕЅ ріЦріЊ рЅдрЅ▒ рІерѕўрѕ░ріЋ рѕхрѕЮ ріЦріЊ ріарѕГрЅ▓рѕхрЅхріЋ рІФрѕхрЅ│рІЇрЅЃрѕЇрЇб

## рЅБрѕЁрѕфрІФрЅх

- ­Ъјх ріе Instagram Reels рѕЎрІџрЅЃ ріарІЇрЅ│рѕе рѕўрѕерЅЦ
- ­Ъјц рІеріарѕГрЅ▓рѕхрЅх ріЦріЊ рІерѕўрѕ░ріЋ рѕхрѕЮ рѕЏрѕ│рІерЅх
- ­Ъњ┐ рІеріарѕЇрЅарѕЮ рѕўрѕерїЃ рѕЏрѕ│рІерЅх (ріФрѕѕ)
- ­ЪЊЁ рІерѕЁрЅхрѕўрЅх рІЊрѕўрЅх рѕЏрѕ│рІерЅх
- ­Ъј» рІерѕўрЅ░рѕЏрѕўріЋ рѕўрЅХріЏ рѕЏрѕ│рІерЅх
- ­ЪњЙ рЅ│рѕфріГріЋ рЅарІ│рЅ│рЅцрІЮ рІЇрѕхрїЦ рѕЏрѕхрЅђрѕўрїЦ
- ­Ъћњ ріерЇЇрЅ░ріЏ рІ░рѕЁріЋріљрЅх ріЦріЊ рІерІ│рЅ│ рїЦрЅарЅЃ

## рѕўрѕхрЇѕрѕГрЅХрЅй

- PHP 7.4 рІѕрІГрѕЮ ріерІџрІФ рЅарѕІрІГ
- MySQL 5.7 рІѕрІГрѕЮ ріерІџрІФ рЅарѕІрІГ
- FFmpeg (ріерЅфрІ▓рІ« рІхрѕЮрїй рѕЏрІЇрїБрЅх)
- PHP рѕЮрѕхрѕерЅ│рІјрЅй: curl, json, pdo, pdo_mysql
- SSL рІерѕЮрѕхріГрѕГ рІѕрѕерЅђрЅх (рѕѕ webhook)

## рѕўрїФріЋ

### 1. рЇІрІГрѕјрЅйріЋ рѕЏрѕхрїѕрЅБрЅх

рѕЂрѕЅріЋрѕЮ рІерЇЋрѕ«рїђріГрЅх рЇІрІГрѕјрЅйріЋ рЅа cPanel рѕєрѕхрЅхрІј рѕІрІГ рІФрѕхрїѕрЅАрЇб

### 2. рѕўрїФріЋ рѕЏрѕхрЇѕрїИрѕЮ

рѕўрѕхрЇѕрѕГрЅХрЅйріЋ рѕѕрѕЏрѕерїІрїѕрїЦ `install.php` рЅаріарѕ│рѕйрІј рІЇрѕхрїЦ рІФрѕхрЇѕрїйрѕЎрЇб

### 3. рІ│рЅ│рЅцрІЮ рѕЏрІІрЅђрѕГ

1. MySQL рІ│рЅ│рЅцрІЮ рІГрЇЇрїарѕЕ
2. `config.php` рІФрѕГрЅхрІЉ ріЦріЊ рІерІ│рЅ│рЅцрІЮ рѕўрѕерїЃ рІФрѕхрїѕрЅА

### 4. рЅ┤рѕїрїЇрѕФрѕЮ рЅдрЅх рѕўрЇЇрїарѕГ

1. рЅарЅ┤рѕїрїЇрѕФрѕЮ рІЇрѕхрїЦ [@BotFather](https://t.me/botfather) рїІрѕГ рЅ░ріљрїІрїЇрѕЕ
2. `/newbot` рЅхрІЋрІЏрІЮ рІГрѕІріЕ
3. рІерЅдрЅх рѕхрѕЮ ріЦріЊ рІерЅ░рїарЅЃрѕџ рѕхрѕЮ рІГрѕЮрѕерїА
4. рІерЅдрЅх рЅХріеріЋ рІГрЅЁрІ▒
5. рЅХріеріЋріЋ рЅа `config.php` рІЇрѕхрїЦ рІФрѕхрїѕрЅА

### 5. APIs рѕЏрІІрЅђрѕГ

#### Instagram Basic Display API
1. рІѕрІ░ [Facebook Developers](https://developers.facebook.com/) рІГрѕѓрІ▒
2. ріарІ▓рѕх рѕўрЅ░рїЇрЅарѕфрІФ рІГрЇЇрїарѕЕ
3. Instagram Basic Display рІГрїерѕЮрѕЕ
4. API рЅЂрѕЇрЇЇ рІФрїЇріЎ

#### ACRCloud API (рѕЎрІџрЅЃ ріарІЇрЅ│рѕе рѕўрѕерЅЦ)
1. рІѕрІ░ [ACRCloud](https://www.acrcloud.com/) рІГрѕѓрІ▒
2. ріаріФрІЇріЋрЅх рІГрЇЇрїарѕЕ
3. API рЅЂрѕЇрЇЇ рІФрїЇріЎ

### 6. Webhook рѕЏрІІрЅђрѕГ

`setup.php` рІФрѕхрЇѕрїйрѕЎ:

```
https://yourdomain.com/setup.php?password=your_admin_password
```

### 7. рЅдрЅх рѕЏрѕъріерѕГ

```

```
